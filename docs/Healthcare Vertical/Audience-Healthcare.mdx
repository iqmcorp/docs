# Audience API

The Audience API allows you to create different types of Audiences for Healthcare Campaigns. 

The following table describes some of the Audience types:

<div class="definitionsTable">
  | Type | Description |
  | --- | --- |
  | **Matched Audience** | Upload a CSV or XLSX file with NPI numbers which is then matched with their respective digital entities to create a custom Audience segment for healthcare accounts. 
  | **ABM Audience** | Account-Based Marketing for healthcare Campaigns.
</div>
