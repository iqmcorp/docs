# Migrate an Audience

## Overview 

The API Comparison table highlights the main actions and API methods for creating and managing Audiences. Select an endpoint from IQM or DV360 to review that platform's relevant API documentation. By comparing APIs, you can migrate your workflow from DV360 to IQM.

## Definitions

<th>DV360</th>

<tr>
  <td>Audience Segment</td>
  <td>Audience Segment</td>
  <td>A group of specified Segments with optimization settings.</td>
</tr>
<tr>
  <td>Segment Group</td>
  <td>Segments</td>
  <td>The demographic characteristics by which a group of people can be identified, organized, and combined into Audience Segments (Segment Targeting for DV360).</td>
</tr>
<tr>
  <td>Matched Audience</td>
  <td>Customer Match</td>
  <td>A customized Audience that you've created and uploaded to the platform.</td>
</tr>
<tr>
  <td>Pre-bid Audience</td>
  <td>n/a</td>
  <td>An Audience composed of segments that have been sent to the platform by third-party data providers that you can purchase and target in your Audiences.</td>
</tr>
<tr>
  <td>Geo-Farmed Audience</td>
  <td>Geo Region Targeting</td>
  <td>An Audience created based on a customizable geographic location rather than demographic data.</td>
</tr>
<tr>
  <td>Contextual Audience</td>
  <td>Keyword Targeting</td>
  <td>An Audience generated by keywords or URLs, based on user activity.</td>
</tr>
<tr>
  <td>Retargeted Audience</td>
  <td>Tag Based Audience</td>
  <td>An Audience created by tracking tag conversions that retargets users who visit the tracked site.</td>
</tr>
<tr>
  <td>Campaign Audience</td>
  <td>Activity Based Audience</td>
  <td>An Audience created by users previously targeted by a Campaign.</td>
</tr>
