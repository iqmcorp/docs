### Validate AQS Report Name

<CopyUrl method="GET" url="https://api.iqm.com/api/v3/ins/aqs-reports/validate/name" />

  <div class="child1">

Validate the report name for generation of AQS Reports. Checks if the name meets length requirements (3-100 characters) and ensures uniqueness within the organization.

| Query Parameters |  |
| ---- | --- |
| `reportName` <br /><span class="type-text">string</span> | Name of report to validate |

  <details className="objectPropertiesDetails" style={{ maxWidth: "100%", padding: "1rem" }}>
    <summary style={{fontSize: "16px"}}>Response Properties</summary>
    <table className="objectProperties">
    <tr>

    </tr>
    <tr>
    <td>`message` <br /><span class="type-text">string</span></td>
    <td>Validation message</td>
    </tr>
    </table>
    </details>

</div><div class="child2">

```json title="Response 200 (valid)"
{
  "success": true,
  "data": {
    "available": true,
    "message": "This AQS report name is valid"
  }
}
```

```json title="Response 200 (already used)"
{
  "success": true,
  "data": {
    "available": false,
    "message": "The name 'report_name' is already used by another report, kindly update the name"
  }
}
```

```json title="Response 422"
{
  "success": false,
  "errorObjects": [
    {
      "error": "Please provide name with at least 3 characters and maximum of 100 characters"
    }
  ]
}
```

### Generate AQS Report

<CopyUrl method="POST" url="https://api.iqm.com/api/v3/ins/aqs-reports" />

<div class="endpoint-container">
  <div class="child1">

Generate an Audience Quality Score (AQS) Report. This API calculates the cost for the impressions based on the impressions and mark up charged on the Organization and then creates the AQS Reports.

| Request Schema |  |
| ---- | --- |
| `campaignId` <br /><span class="type-text">integer</span> | [Campaign ID](/guidelines/campaign-api#get-list-of-campaigns) |
| `aqsId` <br /><span class="type-text">integer</span> | [AQS Report ID](#get-list-of-aqs-reports) to regenerate |
| `aqsName` <br /><span class="type-text">string</span> | AQS Report name |
| `aqsStartDate` <br /><span class="type-text">integer</span> | Unix epoch timestamp of start date, in seconds |
| `aqsEndDate` <br /><span class="type-text">integer</span> | Unix epoch timestamp of end date, in seconds |
| `aqsReportTypeId` <br /><span class="type-text">integer</span> | [AQS Report Type ID](#get-aqs-report-types) |
| `aqsAudienceConceptGroups` <br /><span class="type-text">array of objects</span> | Array of audience/concept group mappings |

<a href="#aqs-resource-properties"><button className="responsePropertiesButton"><Caret />Resource Properties</button></a>
