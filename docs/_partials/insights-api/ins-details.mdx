### Get Eligible Campaigns

<CopyUrl method="GET" url="https://api.iqm.com/api/v3/ins/insights/eligible-campaigns" />

<div class="endpoint-container">
  <div class="child1">

Get a status-wise list of eligible Campaigns for generation of PLD, ICT, or NLD Insights reports.

Eligible Campaign IDs can be used to in the [Calculate Insights Computation](#calculate-insights-computation) and [Create Report Template](#create-report-template) endpoints.

| `reportType` <br /><span class="type-text">string</span> <span class="required-text">required</span> | Type of report: <var>pld</var>, <var>ict</var>, or <var>nld</var> |

### Calculate Insights Computation

<CopyUrl method="POST" url="https://api.iqm.com/api/v3/ins/insights/computation" />

<div class="endpoint-container">
  <div class="child1">

Compute the cost and impressions for creation of PLD, ICT, or NLD Insights reports based on the provided Campaign IDs and date range.

The computation calculates the chargeable impressions and cost based on the markup charged to the organization.

| Request Schema |  |
| ---- | --- |
| `category` <br /><span class="type-text">string</span> <span class="required-text">required</span> | Insight type category: <var>pld</var>, <var>ict</var>, or <var>nld</var> |
| `campaignIds` <br /><span class="type-text">array of integers</span> <span class="required-text">required</span> | [Campaign IDs](/guidelines/campaign-api#get-list-of-campaigns) for which to compute the report |
| `startDate` <br /><span class="type-text">integer</span> <span class="required-text">required</span> | Unix epoch timestamp of report start date, in milliseconds |
| `endDate` <br /><span class="type-text">integer</span> <span class="required-text">required</span> | Unix epoch timestamp of report end date, in milliseconds |

import {
  getInstance
} from "prerequisites"

const axios = getInstance();

interface Responses {
  200: {
    content: {
      "application/json": {
        success: boolean;
        data: {
          ioId: number;
          ioName: string;
          startDate: number;
          endDate: number;
          campaignIds: number[];
          reportCreatable: boolean;
          chargeableImps: number;
          chargedImps: number;
          chargeableCost: number;
          chargedCost: number;
          fundsAvailable: boolean;
          effectiveRate: number;
          templateId: number;
        }
      }
    };
  };
}

function computeInsightsReports(): Promise<Responses> {
  const options = {
    method: 'POST',
    url: 'https://api.iqm.com/api/v3/ins/insights/computation',
    requestBody: {
      content: {
        "application/json": {
          category: `string`,
          campaignIds: `number[]`,
          startDate: `number`,
          endDate: `number`,
        }
      }
    }
  };

  return axios.request(options).then(({ data }: { data: Responses }) => data);
}
