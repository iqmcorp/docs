### Create Pre-bid Audience

<CopyUrl method="POST" url="https://api.iqm.com/api/v3/audience/prebid" />

<div class="endpoint-container">
  <div class="child1">

Pre-bid Audiences are groups of [Fraud , Brand Safety, and Viewability<ExternalLink />](https://help.iqm.com/en/articles/8698600-fraud-brand-safety-and-viewability-pre-bid-targeting-overview) pre-bid segments that you select. You can create these audiences and seamlessly integrate them into new or existing Campaigns. 


:::warning

The IQM platform supports one Creative type per Campaign. Pre-bid segments include both **display** and **video** that can be selected. Choose only one format for each Pre-bid Audience to avoid incurring extra charges. 

:::

| --- | --- |
| `audienceName` <br /><span class="type-text">string</span> | Pre-bid Audience name |
