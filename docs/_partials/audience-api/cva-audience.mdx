### Get Custom Voter Audience Segment Details

<CopyUrl method="GET" url="https://api.iqm.com/api/v3/audience/cva/segments/{segmentId}" />

<div class="endpoint-container">
  <div class="child1">

Get Custom Voter Audience segment details by segment ID.

This endpoint will fetch the segments based on search parameters provided for Custom Voter Audience segment catalogue.
  * If no search parameters are provided then it will fetch all root segments for whole country (currently U.S.) along with their child segments.
  * If state IDs are provided then it will fetch all root segments for those states along with their child segments.
  * If parent segment ID is provided then it will fetch all child segments for that parent segment.
  * If search keywords are provided then it will fetch all leaf segments matching those keywords in column path_key and description.
  * If taxonomy IDs are provided then it will fetch all leaf segments matching those taxonomy Ids. If validateTaxonomyIDs flag is true then it will validate all taxonomy IDs exist.
  * If segment IDs are provided then it will fetch those segments.
  * If none of the above criteria is met then it will return empty list.

| Path Parameter | |
| --- | --- |
| `segmentId` <br /><span class="type-text">integer</span> | [Segment ID](/guidelines/audience-api/#search-segmented-audiences) |

| Query Parameters | |
| --- | --- |
| `stateIds` <br /><span class="type-text">array of integers</span> | List of state IDs used to fetch root segments for those states along with their child segments |
| `parentSegmentId` <br /><span class="type-text">integer</span> | Parent segment ID used to fetch child segments for that parent |
| `searchField` <br /><span class="type-text">string</span> | Comma-separated search keywords used to fetch leaf segments matching those keywords in columns path_key and description |
| `taxonomyIds` <br /><span class="type-text">array of strings</span> | List of taxonomy IDs used to fetch leaf segments |
| `validateTaxonomyIds` <br /><span class="type-text">boolean</span> | Flag to validate if all taxonomy IDs provided exist, default: <var>false</var> |
| `segmentIds` <br /><span class="type-text">array of integers</span> | List of segment IDs used to fetch segments |
| `pageNo` <br /><span class="type-text">integer</span> | Page number, default: <var>0</var> |
| `noOfEntries` <br /><span class="type-text">integer</span> | Number of entries per page, default: <var>0</var> |

### Create Custom Voter Audience

<CopyUrl method="POST" url="https://api.iqm.com/api/v3/audience/cva" />

<div class="endpoint-container">
  <div class="child1">

Create a Custom Voter Audience by providing audience details in the request body. The response includes the created Audience ID which can be used to retrieve Audience details and apply the Audience to Campaigns.

| Request Body Properties | |
| --- | --- |
| `audienceName` <br /><span class="type-text">string</span> | Audience name |
| `equation` <br /><span class="type-text">string</span> | Equation used to create the Audience based on [Segment IDs](/guidelines/audience-api/#search-segmented-audiences) and boolean operators AND, OR, NOT (represented as &, |, !) (e.g. `(iqm1234&iqm3456)|(iqm21345|!iqm45675)`) |
| `reach` <br /><span class="type-text">integer</span> | Audience reach |
| `audienceSize` <br /><span class="type-text">number</span> | Audience size as a percentage (0-100) |
| `stateIds` <br /><span class="type-text">array of integers</span> | [State IDs](#get-available-states-for-cva-audiences) (Audience will include all states by default if no IDs are included.)|

<details className="objectPropertiesDetails" style={{ maxWidth: "800px", padding: "1rem" }}>
 <summary style={{ fontSize: "16px" }}>Response Properties</summary>
 <table className="objectProperties">
<tr>
  <td>`id` <br /><span class="type-text">integer</span></td>
  <td>Audience ID</td>
</tr>
<tr>
  <td>`message` <br /><span class="type-text">string</span></td>
  <td>Response message</td>
</tr>
</table>
</details>

</div><div class="child2">

```json title="Request Body"
{
  "audienceName": "sampleAudienceName1",

  "reach": 1234566,

  "stateIds": [
    123,
    456
  ]
}
```

```json title="Response 200"

  "success": true,
  "data": {
    "id": 1143140,
    "message": "Custom Voter Audience created successfully"
  }
}
```
