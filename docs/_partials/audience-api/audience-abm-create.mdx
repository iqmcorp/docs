### Upload ABM Filter File

<CopyUrl method="POST" url="https://api.iqm.com/api/v3/audience/abm/healthcare-data/filter-file" />

<div class="endpoint-container">
  <div class="child1">

Extract ABM Healthcare Data filters from an uploaded file. The file is uploaded as multipart/form-data in the <var>abmFilterFile</var> form data field, and filter column mappings and file type are sent as query parameters.

| Request Schema | |
| --- | --- |
| `fileType` <br /><span class="type-text">integer</span> <span class="required-text">required</span> | Type of the file being uploaded: CSV (<var>1</var>), Excel (<var>2</var>) |
| `columns` <br /><span class="type-text">string</span> <span class="required-text">required</span> | Serialized JSON mapping of file column headers to filter field names
| `abmFilterFile` <br /><span class="type-text">multipart/form-data</span> <span class="required-text">required</span> | File containing the ABM audience data to be filtered
