## Cross-Screen Proposal Management

### Split Proposal

<CopyUrl method="POST" url="https://api.iqm.com/api/v3/plan/political/proposals/{proposalId}/split" />

<div class="endpoint-container">
  <div class="child1">

Split an existing Proposal across linear TV and CTV using the provided advanced targeting criteria.

| Path Parameters |  |
| ---- | --- |
| `proposalId` <br /><span class="type-text">integer</span> <span class="required-text">required</span> | [Proposal ID](#get-list-of-proposals) to split |

| Request Schema |  |
| ---- | --- |
| `splitTypeId` <br /><span class="type-text">integer</span> | [Split Type ID](#get-split-types) |
| `locations` <br /><span class="type-text">array of integers</span> | [Targeting Type IDs](#get-targeting-types) for locations |
| `devices` <br /><span class="type-text">array of integers</span> | [Device Type IDs](/guidelines/master-api/#get-device-type) |
| `inventoryGroupIds` <br /><span class="type-text">array of integers</span> | [Inventory Group IDs](/guidelines/inventory-api/#get-inventory-groups-list) |
| `audienceTypes` <br /><span class="type-text">array of integers</span> | Audience Type IDs |
| `creatives` <br /><span class="type-text">array of integers</span> | [Creative Type IDs](/guidelines/creative-api/#get-creative-types-list) |

<details className="objectPropertiesDetails" style={{ maxWidth: "800px", padding: "1rem" }}>
  <summary style={{ fontSize: "16px" }}>Response Properties</summary>
<table className="objectProperties">
<tr>
  <td>`id` <br /><span class="type-text">integer</span></td>
  <td>[Proposal ID](#get-list-of-proposals)</td>
</tr>
<tr>
  <td>`message` <br /><span class="type-text">string</span></td>
  <td>Confirmation message</td>
</tr>
</table>
</details>

</div>
<div class="child2">

```json title="Request Sample"
{
  "advanced": {
    "locations": [15]
  }
}
```

```json title="Response 200"
{
  "success": true,
  "data": {
    "id": 42,
    "message": "Proposal updated successfully."
  }
}
```

</div></div>
