# Partnerships & Integrations

IQM partners with leading data, measurement, reporting, and supply partners to provide robust targeting, performance management, and safe, transparent media placements.

The following guides help you connect IQM with third-party platforms for reporting, data visualization, and workflow automation.

For a complete list of IQM's data, measurement, reporting, and supply partners, visit the [IQM Partnerships Page<ExternalLink />](https://iqm.com/partnerships/).
