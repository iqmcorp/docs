# Google Looker Studio Integration

Connect IQM Reports to Google Looker Studio to create powerful data visualizations and dashboards from your campaign data.

## About Google Looker Studio

Looker Studio (formerly Google Data Studio) is a complete enterprise business intelligence suite that helps you go beyond dashboards and infuse your workflows and applications with the intelligence needed to make data-driven decisions.

## Prerequisites

- Active IQM account with API access
- Google account with access to Looker Studio
- Valid IQM API credentials
