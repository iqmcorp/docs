# HubSpot Integration

Create matched audiences using HubSpot contact lists to enhance your targeting capabilities in IQM.

## About HubSpot

HubSpot is the customer platform that helps businesses connect and grow better. HubSpot delivers seamless connection for customer-facing teams with a unified platform that includes AI-powered engagement hubs, a Smart CRM, and a connected ecosystem with over 1,500 App Marketplace integrations.

## Prerequisites

- Active IQM account with API access
- HubSpot account with contact lists
- Valid API credentials for both platforms

## Use Cases

- **Retargeting campaigns**: Target existing customers from your HubSpot CRM
- **Lookalike audiences**: Build lookalike audiences based on your best customers
- **Exclusion lists**: Exclude existing customers from prospecting campaigns
- **Cross-sell campaigns**: Target specific customer segments for upsell opportunities

## Key Features

- **Direct list sync**: Import HubSpot contact lists as matched audiences
- **Audience updates**: Keep your audiences fresh with regular syncs
- **Flexible targeting**: Combine HubSpot audiences with IQM's targeting options
