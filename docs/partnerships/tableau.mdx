# Tableau Integration

Tableau is a data visualization software used to create visual reports and dashboards from data sets imported from databases or the web. The IQM Reports web data connector (WDC) for Tableau allows users to import IQM Campaign ad-serving data into Tableau and create visual reports for data analysis.

## Prerequisites

Before using the IQM Reports web data connector, you must have:
- An active IQM account
- Tableau Desktop or Tableau Public installed
  - [Tableau Public<ExternalLink />](https://public.tableau.com/en-us/s/) (free)
  - [Tableau Desktop<ExternalLink />](https://www.tableau.com/products/desktop)

:::info[note]
The IQM Reports data connector is compatible with Tableau Desktop version 2021.1.2 or later.
:::

If you are unfamiliar with Tableau, detailed documentation can be found at [Tableau Help<ExternalLink />](https://help.tableau.com/current/pro/desktop/en-us/default.htm).

## Setup

### Step 1: Open Tableau

Open Tableau Desktop or Tableau Public.

### Step 2: Connect to Web Data Connector

1. On the landing screen, select **More** under **Connect > To a Server** in the sidebar
2. Select **Web Data Connector**

<center><img className='diagram' src={Step1Wdc} /></center>

### Step 3: Load the IQM Reports Connector

1. Type `https://tableau-connector.iqm.com/` into the address bar
2. Press **Enter** to load the IQM Reports web data connector

### Step 4: Log In

Enter your IQM email and password and click the **Log In** button.

<center><img className='diagram' src={Step2Login} /></center>

### Step 5: Configure Report Parameters

Input the parameters for the report data:
- **Timezone**: The timezone for date interpretation
- **Date Range**: The reporting period (begins at midnight on the first day and ends at 11:59 PM on the last day)
- **Group By**: Primary dimension (Campaigns or Creatives)
- **Second Group By**: Secondary dimension (optional)

<center><img className='diagram' src={Step3Parameters} /></center>

:::info[note]
The IQM Reports WDC returns ad-serving data only for currently-running Campaigns, and returns tables for both aggregated and daily data.
:::

After filling out the inputs, click **Submit** to continue.

### Step 6: Configure Data Source

1. Tableau takes you to a **Data Source** page
2. Click and drag the desired table from the **Table** section in the sidebar
3. The IQM Reports WDC provides both daily and aggregated ad-serving data

<center><img className='diagram' src={Step4Tables} /></center>

<center><img className='diagram' src={Step5DataSource} /></center>

For detailed information about Tableau data sources, see the [Official Tableau Documentation<ExternalLink />](https://help.tableau.com/current/pro/desktop/en-us/datasource_prepare.htm).

### Step 7: Preview Data

Click **Update Now** or **Update Automatically** in the bottom-right panel to preview the data.

## Creating Visualizations

### Worksheets, Dashboards, and Stories

- **Worksheet**: A single chart or visualization
- **Dashboard**: A collection of worksheets
- **Story**: A sequence of worksheets and dashboards

To create a new worksheet, dashboard, or story, click on the icons in the bottom left of the window.

### Building a Visualization

1. Click on **Sheet 1** in the bottom left corner to open the Tableau Workspace
2. Values from the IQM Reports WDC are listed in the left sidebar
3. Drag fields from the sidebar to:
   - The areas that say "Drop field here"
   - The **Columns** and **Rows** shelves at the top

### Understanding Data Types

Tableau distinguishes between:
- **Dimensions**: Qualitative values (names, dates)
- **Measures**: Quantitative numeric values

And between:
- **Discrete** (blue): Adds headers to the view
- **Continuous** (green): Adds axes to the visualization

Right-click on any field to convert between discrete and continuous.

### Example Visualization

To create a chart showing clicks and impressions by Campaign:
1. Drag **Campaign Name** to the Columns shelf
2. Drag **Clicks** and **Impressions** to the Rows shelf

Tableau automatically creates an appropriate chart based on your data selections.

For detailed information on creating different visualization types, see [Tableau Building Visualizations<ExternalLink />](https://help.tableau.com/current/pro/desktop/en-us/building_overview.htm).

## Using Multiple Tables

To use multiple tables with distinct Group By dimensions from IQM Reports:

1. Select **Data > New Data Source** from the top menu bar
2. Select **Web Data Connector**
3. Enter the IQM Reports WDC URL and configure new parameters
4. Drag a table to the Data Source editor

<center><img className='diagram' src={Step6MultipleTables} /></center>

<center><img className='diagram' src={Step7Relationship} /></center>

You can switch between data sources using the data source cylinder icon, and in each worksheet, switch between data sources in the **Data** tab of the left sidebar.

## Use with Tableau Server

To upload data extracts to Tableau Server, your administrator must first allow web data connectors and whitelist the IQM Reports connector.

<center><img className='diagram' src={ServerConfig} /></center>

### Administrator Setup

Run the following commands (this will restart the server):

```bash
tsm data-access web-data-connectors allow -r true
tsm data-access web-data-connectors add --url https://tableau-connector.iqm.com:443 -n "IQM Web Data Connector"
tsm pending-changes apply
```

<center><img className='diagram' src={ServerWdc} /></center>

### Publishing to Tableau Server

1. In Tableau Desktop, select **Server > Publish Data Source > IQM Reports**
2. Set **Authentication** to **Embedded Password**
3. Click **Publish**

<center><img className='diagram' src={ServerExtract} /></center>

<center><img className='diagram' src={ServerSchedule} /></center>

:::info[note]
Extracts created from the IQM Reports connector can be refreshed for up to four weeks from the time the data source was created.
:::

For more information on using web data connectors in Tableau Server, see the [Official Tableau Documentation<ExternalLink />](https://help.tableau.com/current/server/en-us/datasource_wdc.htm).
