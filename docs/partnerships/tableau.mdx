# Tableau Integration

Visualize IQM data with Tableau's powerful analytics platform for advanced data exploration and insights.

## About Tableau

The Tableau platform is known for taking any kind of data from almost any system and turning it into actionable insights with speed and ease. It's as simple as dragging and dropping. Plus, Tableau's industry-leading enablement resources, training, and global data community offer unparalleled support for analytics investments.

## Prerequisites

- Active IQM account with API access
- Tableau Desktop or Tableau Online/Server account
- Valid IQM API credentials
