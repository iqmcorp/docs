# Supermetrics Integration

Integrate IQM data into data warehouses, BI tools, and spreadsheet applications with Supermetrics.

## About Supermetrics

Supermetrics specializes in marketing data integration tools that make it easy to move and integrate data to data warehouses, BI tools, and easy-to-use applications such as Google Data Studio, Google Sheets, and Microsoft Excel.

## Prerequisites

- Active IQM account with API access
- Supermetrics account
- Valid IQM API credentials

## Key Features

- **Multiple destinations**: Send IQM data to spreadsheets, BI tools, or data warehouses
- **Scheduled data pulls**: Automate data refresh at specified intervals
- **Custom queries**: Select specific metrics and dimensions for your reports
- **Data blending**: Combine IQM data with other marketing data sources

## Supported Destinations

- Google Sheets
- Google Looker Studio
- Microsoft Excel
- BigQuery
- Snowflake
- And more...
