# Microsoft Power BI Integration

Connect IQM Reports to Microsoft Power BI for enterprise business intelligence dashboards and data visualization.

## About Microsoft Power BI

Power BI is a unified, scalable platform for self-service and enterprise business intelligence (BI). Connect to and visualize any data, and seamlessly infuse the visuals into the apps you use every day.

## Prerequisites

- Active IQM account with API access
- Microsoft Power BI Desktop or Power BI Service account
- Valid IQM API credentials
