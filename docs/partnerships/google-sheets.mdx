# Google Sheets Integration

Pull IQM reporting data directly into Google Sheets for easy data analysis and collaboration.

## About Google Sheets

Google Sheets is a cloud-based spreadsheet tool that allows users to create, edit, and collaborate on data in real-time. It supports various functions, including calculations, data visualization (charts, graphs), and automation.

## Prerequisites

- Active IQM account with API access
- Google account with access to Google Sheets
- Valid IQM API credentials
