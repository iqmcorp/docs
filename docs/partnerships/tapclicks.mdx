# TapClicks Integration

Connect IQM Reports to TapClicks for unified marketing analytics and operations management.

## About TapClicks

TapClicks Smart Marketing Cloud offers an AI-enabled operations and data management platform which includes over 7,500 MarTech/AdTech connectors, automated warehousing, scalable reporting, and data analytics. It is enhanced on the operations side by sales enablement and order, workflow and project management. Thousands of agencies, media companies, and brands leverage TapClicks to deliver the best results for their customers and stakeholders.

## Prerequisites

- Active IQM account with API access
- TapClicks account
- Valid IQM API credentials
