# Zapier Integration

Zapier is an online tool used to connect and automate workflows between web applications. The IQM Reports app provides a convenient way to use Zapier to connect your IQM ad-serving data to other web applications via automated workflows, or Zaps.

A Zap allows you to chain web apps—such as Slack, Google Calendar, and Dropbox—together, so that when a specified event occurs in an app, Zapier performs a workflow that passes information between and executes actions in one or more other apps.

## Prerequisites

Before using the IQM Reports Zapier app, you must have:
- An active IQM account
- A Zapier account

## Setup

### Step 1: Log In to Zapier

Go to [Zapier<ExternalLink />](https://zapier.com/) and log in using your Zapier account.

<center><img className='diagram' src={Step1Dashboard} /></center>

### Step 2: Create a New Zap

From the Dashboard, click on **Create Zap**. This will open the Zap editor.

<center><img className='diagram' src={Step2CreateZap} /></center>

### Step 3: Configure the Trigger

In the Trigger section of the Zap, select the Zap's trigger—the app and event that will cause the workflow to execute.

For example, you can use the **Schedule by Zapier** trigger, which causes the Zap to run at specific time intervals.

<center><img className='diagram' src={Step3Trigger} /></center>

For more information about Zap triggers, see [Zapier's Trigger Documentation<ExternalLink />](https://zapier.com/help/create/basics/set-up-your-zap-trigger).

### Step 4: Fill Out Trigger Settings

Fill out the Trigger section to specify trigger settings. For the Schedule by Zapier trigger, you will be asked to specify:
- The time interval in which to periodically run the Zap workflow
- The specific date or time to run the Zap

When done filling out each page of the Trigger, click **Continue** to move to the next section or page.

<center><img className='diagram' src={Step4Schedule} /></center>

### Step 5: Test the Trigger

Test the trigger by clicking the **Test Trigger** button. This allows Zapier to verify that the trigger is working correctly.

When you test the trigger, the output fields of the trigger are shown (e.g., `id`, `pretty_date`, `pretty_time`). The output fields of the trigger step can be mapped into the input fields of any subsequent action in the Zap.

<center><img className='diagram' src={Step5Test} /></center>

### Step 6: Add the IQM Reports Action

Choose and configure the Action that the Zap performs each time it is triggered:

1. Search for **"IQM Reports"** in the search bar
2. Select the **IQM Reports** option with the green "Latest" icon next to it

<center><img className='diagram' src={Step6IqmAction} /></center>

### Step 7: Select the Action Event

From the Action Event dropdown, select **Generate CSV Report**. This action generates a CSV report containing the aggregated ad-serving data of your running Campaigns generated over a specified date range.

Click **Continue**.

<center><img className='diagram' src={Step7CsvAction} /></center>

### Step 8: Sign In to IQM

Click the **Sign in to IQM Reports** button. In the popup:
1. Fill in your IQM email address and password
2. Click **Yes, Continue**
3. Click **Continue** to set up the action

<center><img className='diagram' src={Step8Signin} /></center>

### Step 9: Configure Report Settings

Set up the action by choosing:

| Parameter | Description |
| --- | --- |
| **Time Zone** | The timezone for the report date range |
| **Time Interval** | Predefined intervals (start at midnight on the first day and end at 11:59 PM on the last day) |

<center><img className='diagram' src={Step9Parameters} /></center>

#### Custom Time Interval

The **Custom Time Interval** option allows for finer control over the report's date range:
- Input specific dates and times
- Map datetime values from the trigger into the start and end dates
- Use Zapier's built-in datetime modifiers

<center><img className='diagram' src={Step10CustomInterval} /></center>

:::info[note]
Any timezone offsets entered in custom datetime fields are ignored by the IQM Reports app. The time zone selected in the dropdown is used to calculate the report date range.
:::

For information on datetime modifiers, see [Zapier's Datetime Documentation<ExternalLink />](https://zapier.com/help/create/customize/adjust-date-and-time-values-in-zaps).

### Step 10: Add Additional Actions

After setting up IQM Reports, you can add further actions by clicking the **+** symbols. This allows you to make use of the CSV ad-serving report file generated by IQM Reports by mapping it into other apps, such as Dropbox or Slack.

## Example: Posting Reports to Slack

### Add Slack Action

1. Select **Slack** as the next action following IQM Reports
2. Choose the **Send Channel Message** action event
3. Sign into your Slack account when prompted

<center><img className='diagram' src={Step11SlackAction} /></center>

### Configure Slack Message

Fill out the required fields:
- **Channel**: Choose the Slack channel to post the message to
- **Message Text**: Fill in the message text (you can map output fields from IQM Reports, such as Date Range Start and Date Range End)
- **File**: Map the **Report File** from IQM Reports to attach the CSV report to the Slack message

<center><img className='diagram' src={Step12SlackConfig} /></center>

<center><img className='diagram' src={Step13FileAttach} /></center>

Click **Continue**, then test the action. You should see a message appear in the selected Slack channel with the CSV report attached.

### Turn On the Zap

Click the **Turn on Zap** button to activate your automation.

## Formatting Date Outputs

The Date Range Start and Date Range End output fields are in ISO-8601 datetime format. To display these dates in a more readable format, use the **Formatter by Zapier** action:

1. Click the **+** button above the Slack action
2. Select **Format** for the action
3. Under Choose app & event, select **Date / Time** in the Action Event dropdown and **Format** in the Transform dropdown

<center><img className='diagram' src={Step14Formatter} /></center>

### Configure the Formatter

| Field | Value |
| --- | --- |
| **Input** | Map the Date Range Start output field from IQM Reports |
| **To Format** | Choose an output format from the dropdown |
| **From Format** | Choose Custom and enter: `YYYY-MM-DDTHH:mm:ss.SSSSZ` |
| **From Timezone** | Same timezone as IQM Reports |
| **To Timezone** | Same timezone (or different to convert) |

For more information about datetime patterns, see [Zapier's Datetime Formatting Guide<ExternalLink />](https://zapier.com/help/create/format/modify-date-and-time-formats-in-zaps).
