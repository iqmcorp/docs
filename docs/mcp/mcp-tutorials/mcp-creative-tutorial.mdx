# MCP Creative Tutorial

Use the IQM MCP to interact with the Creative app.

## Overview

This tutorial will guide you through using the IQM MCP to access and manage Creatives in the Creative app. You will learn how to retrieve Creative details, upload new Creatives, and update existing Creatives using the MCP interface.

## Before You Begin 

To upload a Creative using the IQM MCP the following are required:

1. An Account On the IQM Platform
   *  See [Getting Started](/getting-started/before-you-begin) section to create an account and request a Client ID and Client Secret
1. Image, Video, Audio, HTML, VAST, DAAST, CSV, or XLSX File to Upload Creative
1. An extension that allows Claude to interact with your local filesystem, enabling it to read and write files directly.

<div style={{ maxWidth: "1000px" }}>
:::info 

**File requirements for Creative file:**
  * 2MB for JPG, JPEG, PNG
  * 750KB for GIF
  * 400MB for MOV, MP4
  * 100MB for MP3, WAV, OGG, MPEG

For more information on [VAST<ExternalLink />](https://www.iab.com/guidelines/vast/) and DAAST XML format specifications, please see [IAB Tech Lab's Documentation<ExternalLink />](https://iabtechlab.com/standards/digital-audio-ad-serving-template-daast/).

Sample CSV file: [https://app.iqm.com/creatives/example-files/sample_html_creatives.csv<ExternalLink />](https://app.iqm.com/creatives/example-files/sample_html_creatives.csv)

:::
</div>

## Enable Filesystem Extension

To upload a Creative file using the IQM MCP, you will need to enable an extension in Claude that allows the AI model to read and write files directly from your local machine.

To enable the Filesystem extension in Claude navigate to `Settings > Extensions > Browse Extensions` and search for "Filesystem". Click "Install" to add the extension to Claude. 

The extension will prompt you to choose a directory on your local machine that Claude can access. Select a directory where you would like to store and manage your Creative files.
