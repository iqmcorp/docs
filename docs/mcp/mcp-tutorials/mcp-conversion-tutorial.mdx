# MCP Conversion Tutorial

Use the IQM MCP to interact with the Conversion API.

## Overview

This tutorial will guide you through using the IQM MCP to create and manage Conversions with the Conversion API. You will learn how to retrieve Conversion details, create new Conversions, and update existing Conversions using the MCP interface.

## Before You Begin

To create a new Conversion and assign it to a Campaign, the following are required:

1. An Account On the IQM platform
   *  See [Before You Begin](/getting-started/before-you-begin) to create an account and request a Client ID and Client Secret
1. A Draft Campaign
   *  See [Create a Campaign](/quickstart-guides/create-a-campaign-quickstart)


## About IQM Conversions 

IQM uses Conversions as a tool to track user traffic for your Campaigns. 

The IQM platform supports two types of Conversions:
  * **Pixel** Conversions are client-side tracking and they feature advanced settings for fine-tuned optimization.
  * **Postback** Conversions are server-side.

### More Resources
  * [Conversions and Tracking<ExternalLink />](https://help.iqm.com/en/collections/3219750-choose-your-conversion) Help Center article
  * [Conversion API](/guidelines/conversion-api) Guidelines
