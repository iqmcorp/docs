# MCP Finance Tutorial

Use the IQM MCP to interact with the Finance API.

## Overview

This tutorial will guide you through using the IQM MCP to access and manage financial data. You will learn how perform operations in the invoice, credit, and payment lifecycles using the MCP interface.

## Before You Begin

To access financial data using the IQM MCP the following are required:

1. An Account On the IQM Platform
   *  See [Getting Started](/getting-started/before-you-begin) section to create an account and request a Client ID and Client Secret

## About IQM Finances

IQM's finance tools allow you to manage your organization's and its associated customers' finances, including invoicing, payments, and credits.

The workspace owner can create invoicing templates, approve invoices, add payments on behalf of customers, and offer credits to customers. Customers can view their invoices, make payments, and claim offered credits.
