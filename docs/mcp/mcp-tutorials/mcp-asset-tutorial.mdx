# MCP Assets Tutorial

## Overview

This tutorial will guide you through using the IQM MCP to access and manage Assets with the Asset API. You will learn how to retrieve Asset details, upload new Assets, and update existing Assets using the MCP interface.

## Before You Begin

To upload a Asset using the IQM MCP the following are required:

1. An Account On the IQM Platform
   *  See [Getting Started](/getting-started/before-you-begin) section to create an account and request a Client ID and Client Secret
1. A file to Upload as an Asset (Image, Video, Audio, Document, or ZIP)

## About IQM Assets

The Asset API streamlines the management and storage of media files. This service plays a critical role in handling media uploads, temporary storage, and coordination with other services for final storage and metadata management.
