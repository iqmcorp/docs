# MCP Workspace Tutorial

Use the IQM MCP to interact with the Workspace API.

## Overview

This page provides examples of how to use the IQM MCP to access and manage workspaces with the Workspace API. The Workspace API is used to manage Organizations, Workspaces, Customers, and Advertisers:

  * **Organizations** are top-level descriptions of a company's profile, including its name, website, and location details. 
  * **Workspaces** operate within **Organizations** and can be **Customers** or **Advertisers**. 
  * **Users** operate within **Workspaces** and can be assigned **Customers**.

## Before You Begin

To access workspace API tools using the IQM MCP, the following are required:

1. An Account On the IQM Platform
   *  See [Before You Begin](/getting-started/before-you-begin) to create an account and request a Client ID and Client Secret
1. User must have login access to the Organization and access to the Customer app to perform Customer-related actions
