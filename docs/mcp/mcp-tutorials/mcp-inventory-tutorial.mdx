# MCP Inventory Tutorial

Use the IQM MCP to interact with the Inventory API.

## Overview

This tutorial will guide you through using the IQM MCP to retrieve and manage inventory details for your Campaigns. You will learn how to create and manage deals, assign them to Campaigns, and create inventory groups using the MCP interface.

## Before You Begin

To create a deal or inventory group, the following are required:

  * An Account On the IQM platform
    * See [Getting Started](/getting-started/before-you-begin) section to create an account and request a Client ID and Client Secret
  * A Campaign
    * See [Create a Campaign](/quickstart-guides/create-a-campaign-quickstart) Quickstart Guide
    * See [Create a PG Campaign](/tutorials/create-a-pg-campaign) Tutorial
    * Required for assigning a deal/group to a Campaign

## About IQM Inventory

The Inventory API allows you to manage inventory details for your Campaigns, including creating and managing deals, assigning them to Campaigns, and creating inventory groups.
