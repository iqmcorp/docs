# MCP Campaign Tutorial

Use the IQM MCP to interact with the Campaign API.

## Overview

This tutorial will guide you through using the IQM MCP to access and manage Campaigns and Insertion Orders (IOs) with the Campaign API.

The Campaign MCP supports a limited set of features of the Campaign API. For a comprehensive list of supported features and best practices for using the Campaign API, please refer to the [Campaign API Guidelines](/guidelines/campaign-api).

## Before You Begin

To create an IO and Campaign, the following are required:

1. An Account On the IQM Platform
   *  See [Getting Started](/getting-started/before-you-begin) section to create an account and request a Client ID and Client Secret
1. An Approved and Running Creative 
   * See [MCP Creative Tutorial - Upload a Creative](/mcp/mcp-tutorials/mcp-creative-tutorial/#upload-a-creative)
1. A Conversion
   * See [MCP Conversion Tutorial - Create a Conversion](/mcp/mcp-tutorials/mcp-conversion-tutorial/#create-a-conversion)
1. A PG Deal (for PG Campaigns)
   * See [Create a Deal](/tutorials/deal-guide) Tutorial

## About IQM Campaigns

IQM's Campaigns define the individual marketing strategies of your ads. By specifying the supported parameters you can use the API to create a new Campaign.
