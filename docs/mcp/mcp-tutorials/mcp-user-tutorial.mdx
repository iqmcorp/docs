# MCP User Tutorial

Use the IQM MCP to interact with the User Management API. 

## Overview

This page provides instructions on using the IQM MCP to perform operations related to user management, such as logging in and out, resetting or updating passwords, and updating user details such as profile information and status. 

### More Resources
 * [Sign Up and Authenticate](/quickstart-guides/authentication-quickstart-guide/)
 * [User Management API](/guidelines/user-api/)
 * [Account Setup and Management<ExternalLink />](https://help.iqm.com/en/collections/14628352-account-setup-and-management) Help Center articles
