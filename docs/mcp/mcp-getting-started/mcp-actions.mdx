## Bid Model 

IQM's Bid Model app allows you to create and manage bid models for your advertising campaigns. You can define various parameters and strategies to optimize your bids and improve campaign performance.

More resources:
* [Bid Model API](/guidelines/bid-model-api/)
* [Create a Bid Model](/tutorials/create-a-bid-model/) Tutorial
* [Campaign Delivery and Optimization<ExternalLink />](https://help.iqm.com/en/collections/3156335-campaign-delivery-and-optimization)

## Dashboard

IQM's Dashboard provides an at-a-glance overview of your Campaigns' performance. The Dashboard allows you to see all of your Campaigns and Insertion Orders, make adjustments to their budgets, and see customized analytics.  

More resources:
* [Dashboard API](/guidelines/dashboard-api/)
* [Data Dashboards<ExternalLink />](https://help.iqm.com/en/collections/11510700-data-dashboards) Help Center Article
