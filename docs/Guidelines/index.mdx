# API Guidelines

Explore the common endpoints and methods associated with each of IQM's applications.

The Guidelines pages provide overviews of each application and comperehensive coverage of each endpoint supplemented with request and response samples in JSON and TypeScript. See [TypeScript Prerequisites](/Getting%20Started/Typescript-Prerequisites.mdx) for more information on how to make TypeScript API calls.

Download the [OpenAPI Specification JSON File](./assets/openapi/openapi.json).
