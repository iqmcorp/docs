# Setup Guide

## Overview 

IQM MCP is a powerful interface that allows AI models to access information and functionality of IQM's various apps. Use this set up guide to get up and running with IQM MCP. 

## Before You Begin

Before you can use the IQM MCP, you will need an account on the [IQM Platform](https://app.iqm.com/). If you do not have an account, reach out to IQM Support (support@iqm.com) or your IQM representative to get started.

### Authorization Credentials 

You will need the following credentials to connect to IQM's MCP servers:
* Organization Workspace ID
* Bearer Token

##### OW ID 

To find your Organization Workspace ID (OW ID):
1. Log in to the [IQM Platform](https://app.iqm.com/)
2. Click on your profile icon in the top right corner
3. Select "Manage my profile"
4. Navigate to "Organizations" tab
5. Your OW ID is listed on the User's Org card 

<center><img className='diagram' src={MCPowid} /></center>

##### Bearer Token 

To find your Bearer Token:
1. While logged in to the [IQM Platform](https://app.iqm.com/), right click anywhere on the page and select "Inspect" to open the Developer Tools.
2. Navigate to the "Application" tab in the Developer Tools.
3. In the left sidebar, under "Storage", click on "Local Storage" and then select the URL for the IQM Platform (https://app.iqm.com).
4. Look for the key named "auth" in the list of local storage items.
5. The value associated with the <var>access_token</var> is your Bearer Token. Copy this value for use in the MCP setup.

<center><img className='diagram' src={MCPBearer} /></center>



## Installation Guide 

### Download Installer 

Download the installer for your operating system:
* [MacOS](/mcp-installer/iqm-mcp-macos.zip)
* [Windows](/mcp-installer/iqm-mcp-windows.zip)

### Install MCP 

The installer contains an executable file to install the IQM MCP. You will be prompted to set up with Claude Desktop or GitHub Copilot. The installer will set up the config files and download the necessary specifications to get started with your chosen AI model. You will also be prompted for your [Authorization Credentials](#authorization-credentials).
