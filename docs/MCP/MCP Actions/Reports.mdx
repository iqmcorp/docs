# Reports

## Overview

The IQM MCP can connect to the Reports app and create Reports. Once a Report is created, it can be executed to retrieve performance data. 

```c title="MCP Server"
http://maas-dashboard.data.iqm/report/mcp
```

### About IQM Reports

IQM's Reports app provides detailed reporting on your Campaigns' performance. You can create customized reports based on various metrics and dimensions to analyze the effectiveness of your advertising efforts.

More resources:
* [Execute a Report](/Guidelines/Reports-API-Guidelines/#execute-a-report)
* [Dimensions and Metrics](/Guidelines/Reports-API-Guidelines/#get-dimensions-and-metrics-details)
