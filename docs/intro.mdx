# IQM API

## Overview

The IQM API is a standard REST API that allows you to develop applications and scripts for interacting with the IQM Platform.

Start using the API with the [Create a Campaign](/Quickstart%20Guides/Upload-Creative-and-Create-a-Campaign-API-Quickstart-Guide.mdx) Quickstart Guide.
