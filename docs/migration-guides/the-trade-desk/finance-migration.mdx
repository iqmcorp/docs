# Migrate Finances

Migrate Finances from The Trade Desk to IQM by comparing each platform's API methods for creating and managing Finances, Invoices, and Payments.

The API Comparison table highlights the shared actions and API methods between IQM and The Trade Desk for creating and managing Finances, Invoices, and Payments. Select an endpoint to review each platform's relevant API documentation. By comparing APIs, you can migrate your workflow from The Trade Desk to IQM.

For a comprehensive overview of IQM's finance services, see the [Finance API](/guidelines/finance-api) Guidelines.
